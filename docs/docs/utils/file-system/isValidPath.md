# isValidPath

Checks if a `directory` exists at the given `file path`.

- import the method

```js
import { isValidPath } from "nodelpers";
```

- params

```js
fp: string;
```

- usage

```js
let path = "/path/to/your/dir";
isValidPath(path); // true or false
```
