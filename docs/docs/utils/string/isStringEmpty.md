# isStringEmpty

Checks if given string is empty or not.

- import the method

```js
import { isStringEmpty } from "nodelpers";
```

- params

```js
str: string;
```

- usage

```js
isStringEmpty(""); // true
isStringEmpty(" "); // true
isStringEmpty(" kite"); // false
```
