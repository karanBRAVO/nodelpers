# secondsToHHMMSS

Convert given seconds to `Hours:Minutes:Seconds` format.

- import the method

```js
import { secondsToHHMMSS } from "nodelpers";
```

- params

```js
totalSeconds: number;
```

- usage

```js
secondsToHHMMSS(3600); // 01:00:00
```
