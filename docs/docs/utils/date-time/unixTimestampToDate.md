# unixTimestampToDate

Converts a Unix timestamp (seconds) to a JavaScript `Date` object.

- import the method

```js
import { unixTimestampToDate } from "nodelpers";
```

- params

```js
timestamp: number;
```

- usage

```js
unixTimestampToDate(17009003);
```
