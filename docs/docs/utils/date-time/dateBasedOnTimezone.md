# dateBasedOnTimezone

Converts a given date (or current date) to a specific time zone.

- import the method

```js
import { dateBasedOnTimezone } from "nodelpers";
```

- params

```js
timeZone: string;
```

- usage

```js
dateBasedOnTimezone("Asia/Kolkata");
dateBasedOnTimezone("Asia/Kolkata", "2025-03-11");
```
