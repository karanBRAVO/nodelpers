# HHMMSSToSeconds

Convert given `HH:MM:SS` format to total seconds.

- import the method

```js
import { HHMMSSToSeconds } from "nodelpers";
```

- params

```js
timeString: string;
```

- usage

```js
HHMMSSToSeconds("01:00:00"); // 3600
```
